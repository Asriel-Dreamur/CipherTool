CipherTool — приложение с графическим интерфейсом для создания, сохранения и использования составных алгоритмов шифрования.  
Вы можете комбинировать шесть типов шифров, работать с любыми алфавитами (включая русский) и шифровать текст или файлы.  
Программа полностью автономна — один EXE‑файл без зависимостей.

Возможности:

- **6 типов шифров**: Цезарь, Виженер, замена (substitute), перестановка по ключу (transpose), XOR, разворот строки (reverse).
- **Поддержка алфавитов**: встроенные `rus`, `eng` или любой пользовательский набор символов.
- **Комбинирование шагов**: можно составить цепочку из нескольких шифров, сохранить её как схему и использовать многократно.
- **Текстовый и файловый режимы**: шифруйте сообщения прямо в окне или обрабатывайте любые файлы.
- **Сохранение схем в `.cipher`**: читаемый текстовый формат, который можно редактировать вручную.
- **Редактирование схем**: загруженную схему можно изменить и пересохранить.
- **Полностью автономный EXE**: не требует установки, библиотек или интернета.
- **Адаптивный интерфейс**: окно можно растягивать, элементы подстраиваются.
- **Ассоциация `.cipher`**: двойной клик по файлу схемы открывает программу (регистрируется при первом запуске).
- **Корректная работа с UTF‑8**: все шифры обрабатывают кириллицу и другие многобайтовые символы.

Быстрый старт:

1. Скачайте последнюю версию `CipherToolGUI.exe` из [Releases](https://github.com/Asriel-Dreamur/CipherTool).
2. Запустите программу (никакой установки не требуется).
3. Нажмите **«Создать схему»**, задайте название и алфавит (например, `rus`).
4. Добавьте шаги шифрования (например, `caesar shift=3`).
5. Сохраните схему (имя файла подставится автоматически).
6. На вкладке **«Текст»** введите сообщение и нажмите **«Зашифровать текст»**.
7. Скопируйте результат, вставьте обратно и нажмите **«Расшифровать текст»** — вернётся исходное сообщение.

Сборка из исходников:

Требуется:
- Visual Studio 2022 (или Build Tools) с поддержкой C++17
- CMake ≥ 3.10

```bash
git clone https://github.com/Asriel-Dreamur/CipherTool.git
cd CipherTool
mkdir build && cd build
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Release
